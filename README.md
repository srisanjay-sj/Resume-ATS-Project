# 🚀 Resume ATS Analyzer

An AI-powered full-stack web application that analyzes resumes against job descriptions, providing ATS compatibility scores and actionable improvement suggestions.

🔗 **Live Demo:** https://resume-ats-project.vercel.app  
💻 **Tech Stack:** MERN + Google Gemini AI

---

## ✨ Features

- 🔐 **Secure Authentication** — JWT-based Register/Login
- 📄 **Smart PDF Parsing** — Extracts text from uploaded resumes
- 🎯 **ATS Compatibility Score** — Keyword-based matching against job descriptions
- 🤖 **AI-Powered Analysis** — Google Gemini generates strengths, missing skills, and suggestions
- ✅ **Verdict Badges** — Strong Match / Good Match / Needs Improvement
- 📊 **Analysis History** — View all past resume analyses
- 📥 **PDF Report Export** — Download analysis reports
- 🎨 **Premium Responsive UI** — Dark theme with smooth animations

---

## 🛠️ Tech Stack

**Frontend:** React.js, Vite, React Router  
**Backend:** Node.js, Express.js  
**Database:** MongoDB Atlas  
**AI:** Google Gemini API  
**Deployment:** Vercel (Frontend) + Render (Backend)

## ⚙️ Getting Started

### Backend Setup
\`\`\`bash
cd server
npm install
node server.js
\`\`\`

### Frontend Setup
\`\`\`bash
cd client
npm install
npm run dev
\`\`\`

### Environment Variables (server/.env)
\`\`\`
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_gemini_api_key
\`\`\`
